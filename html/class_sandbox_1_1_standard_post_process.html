Public Attributes

FalloffSettings Falloff = new()
 Fall-Off allows you to adjust the distance of influence for most of the post processing effects. This is used as a modifier to adjust how much influence the post processing effect has based on a start and end distance. This allows you to fade a post-processing effect based on how far away a point is in the world. It's like depth of field but for all post processing effects.
 
PixelateSettings Pixelate = new()
 Pixelate lets you define how many pixels should be drawn on the screen at once. This is identical to scaling your screen down to a specific size and resizing it back to it's original use using a nearest neighbour filter.
 
LensDistortionSettings LensDistortion = new()
 Lens Distortion is used to mainly create a "barrel" distortion or a "fish-eye" effect. Both parameters control how the distorted the image becomes. This is done by making the lens a more concave or a more convex shape
 
PaniniProjectionSettings PaniniProjection = new()
 Panini Projection distorts the view similar to what lens distortion does, however it's distorted in a way to give the feel of having a wider field of view. The sides of the view is distorted to allow for this
 
ChromaticAberrationSettings ChromaticAberration = new()
 Chromatic Aberration is how much the seperate color channels should seperate. As this is in UV space, the color channel seperation should be extremely minimal (close to zero) values.
 
MotionBlurSettings MotionBlur = new()
 Motion blur defines how much an object should blur(based on the current view) depending on how fast it's moving. The faster the user looks around, the more the objects will blur.
 
SharpenSettings Sharpen = new()
 Sharpen brings emphasis on edges and smaller details. As this is increased, the edges and details become more defined in the scene
 
BlurSettings Blur = new()
 Blur does a simple gaussian blur on the current view.
 
DepthOfFieldSettings DepthOfField = new()
 Depth of field is used to emulate an actual camera with focus. It allows the user to keep objects in focus and adjust various other settings to emulate how a film camera works.
 
ColorOverlaySettings ColorOverlay = new()
 Color Overlays allow tinting, mixing or completely changing the color of the current view. This allows for tinting how the world looks or implementation of such effects such as flashing
 
SaturateSettings Saturate = new()
 Saturation defines how saturated a view is. A saturation of 0 leads to a black and white view, whereas a saturation of 1 leads to the regular view.
 
FilmGrainSettings FilmGrain = new()
 Film grain introduces noise to the screen. The noise can be customised to introduce more noise in darker areas. This allows the emulation of a camera sensor.
 
VignetteSettings Vignette = new()
 This creates a vignette or a sort of black border around the the edges of the screen. This is faily used to bring attention to the center of the frame
 
HueRotateSettings HueRotate = new()
 Shift the hue of all the current colors on the screen. This is also known as hue shifting
 
BrightnessSettings Brightness = new()
 Boosting of the brightness or dimming of the brightness of the screen
 
ContrastSettings Contrast = new()
 Ability to increase or decrease the contrast of the screen
