Triggers a haptic vibration event on the controller for this hand.

If a haptic event is already running it will be interrupted immediately.

Parameters
durationHow long the haptic action should last (in seconds - can be 0 to "pulse" it)
frequencyHow often the haptic motor should bounce (0 - 320 in hz. The lower end being more useful)
amplitudeHow intense the haptic should be (0 - 1)
