Detailed Description

WorldInput can be used to simulate standard mouse inputs on WorldPanels.

You need to set P:Sandbox.UI.WorldInput.Ray and P:Sandbox.UI.WorldInput.MouseLeftPressed to simulate inputs, ideally this should be done in a BuildInput event.

You can create this on your Pawn.

A nice way to simulate world inputs for VR is to have a T:Sandbox.UI.WorldInput for each T:Sandbox.Input.VrHand.

The Ray used to intersect with your world panels, simulating mouse position.

This should ideally be set in BuildInput or FrameSimulate.

This could be your P:Sandbox.Local.Pawn EyePos and EyeDir:

WorldInput.Ray = new Ray( Local.Pawn.EyePos, Local.Pawn.EyeDir );

This could also be P:Sandbox.Input.Cursor if you want to use the mouse cursor.

WorldInput.Ray = Input.Cursor;

If using VR you may want to derive the Ray from a T:Sandbox.Input.VrHand.

WorldInput.Ray = new Ray( VRHand.Transform.Position, VRHand.Transform.Rotation.Forward );
WorldInput.MouseLeftPressed = VRHand.Trigger.Active;
